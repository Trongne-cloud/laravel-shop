# Laravel Shop 🛒

Dự án Laravel Shop là một hệ thống bán hàng (e-commerce) được xây dựng bằng framework Laravel.  
Hệ thống hỗ trợ quản lý sản phẩm, danh mục, đơn hàng và người dùng.

---

## 🚀 Tính năng chính

- 👤 Quản lý người dùng (đăng ký / đăng nhập)
- 📦 Quản lý danh mục sản phẩm
- 🧾 Quản lý đơn hàng
- 🖼️ Upload hình ảnh sản phẩm
- 🔎 Hiển thị danh sách sản phẩm và chi tiết sản phẩm
- 🔐 Phân quyền admin / user

---

## 🧰 Công nghệ sử dụng

- Laravel Framework
- PHP 8+
- MySQL / MariaDB
- Bootstrap 5 (UI)
- Blade Template Engine

---

## ⚙️ Yêu cầu hệ thống

- PHP >= 8.3
- Composer
- MySQL
